# 푸시한 다음에 할 일 (차근차근)

`git push -u origin master`까지 끝났다면, 아래 순서대로 진행하면 됩니다.

---

## 1단계: GitHub에서 확인

1. 브라우저에서 **https://github.com/aldas79/GEO_radar** 접속
2. 코드(폴더/파일)가 올라와 있는지 확인
3. 문제 없으면 다음 단계로

---

## 2단계: Windows에서 저장소 받기

1. **Git이 설치된** Windows에서 PowerShell 또는 Git Bash 실행
2. 저장소를 받을 폴더로 이동 (예: 바탕화면, 또는 `C:\Users\본인사용자명\projects`)
3. 아래 명령 실행:

```bash
git clone https://github.com/aldas79/GEO_radar.git
cd GEO_radar
```

4. `GEO_radar` 폴더 안에 `04_geo_data_extension`, `01_playwright_geo_scrna` 등이 보이면 성공

---

## 3단계: Chrome 확장 프로그램으로 테스트

1. Windows에서 **Chrome** 실행
2. 주소창에 **`chrome://extensions`** 입력 후 Enter
3. 우측 상단 **개발자 모드** 켜기
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. **아래 폴더를 선택** (clone한 위치에 맞게 경로만 바꾸세요):
   - `C:\Users\본인사용자명\원하는경로\GEO_radar\04_geo_data_extension\extension`
   - 반드시 **`extension`** 이 들어간 폴더를 선택해야 함
6. 확장이 로드되면 퍼즐 아이콘 옆에 "GEO Download Info Collector" 표시
7. **동작 확인**
   - 확장 아이콘 클릭 → GSE ID에 `GSE138669` 입력
   - **Collect from GEO + SRA + Trace** 클릭
   - 로그에 샘플 수·완료 메시지 나오는지 확인
   - **Download CSV** 클릭 → CSV 파일이 받아지는지 확인

## 4단계: 일상적인 작업 흐름

### Linux 서버에서 코드 수정한 경우

```bash
cd /home/khk/2601_vibecoding/7_week_homework

# 수정 후
git add .
git status
git commit -m "수정 내용 한 줄 요약"

```

### Windows에서 최신 코드 받기

```bash
cd C:\Users\본인사용자명\원하는경로\GEO_radar
git pull
```

- Chrome 확장은 이미 로드된 상태면, **chrome://extensions** 페이지에서 해당 확장의 **새로고침(↻)** 버튼을 누르면 최신 코드로 갱신됨

### Windows에서 수정하고 올리는 경우 (선택)

- Windows 쪽 `GEO_radar` 폴더에서 수정 후:

```bash
cd C:\Users\본인사용자명\원하는경로\GEO_radar
git add .
git commit -m "수정 내용 요약"
git push origin master
```

- 그 다음 Linux 서버에서 받을 때:

```bash
cd /home/khk/2601_vibecoding/7_week_homework
git pull origin master
```
